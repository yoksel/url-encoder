Tentang Tool ini

Kita dapat menggunakan SVG didalam CSS melalaui data URI, tetapi jika tanpa menggunakan encoding, file SVG tersebut hanya bekerja pada Webkit berdasarkan Browser. Jika kita meng-encode SVG menggunakan encodeURIComponent() maka akan bekerja.

SVG harus memiliki atribut xmlns seperti berikut: xmlns='http: //www.w3.org/2000/svg'. Jika tidak ada, maka akan ditambahkan secara otomatis.

SVG yang sudah di Encode dapat digunakan pada background, border-image atau didalam mask (Live Demo).
